# Image Filter App 🖼️

This is a simple Python-based image filter app using **Streamlit**. You can apply filters like grayscale, rotate, and flip to your images.

## 🛠️ Features
- Upload an image
- Apply filters:
  - Grayscale
  - Rotate 90°
  - Flip horizontally
- Display processed image

## 🚀 How to Run

1. Install Streamlit:
   ```bash
   pip install streamlit
streamlit run App.py
